Google 로그인 연동 설정
실제 운영을 위한 OAuth 설정 가이드
Step 1. Google Cloud Console 접속
console.cloud.google.com 접속 → 새 프로젝트 생성
Step 2. OAuth 동의 화면 설정
APIs & Services → OAuth 동의 화면 → 외부 선택 → 앱 정보 입력
Step 3. Client ID 발급
사용자 인증정보 → OAuth 2.0 클라이언트 ID 생성
유형: 웹 애플리케이션 → 대시보드 배포 URL을 리디렉션 URI에 추가
Step 4. 이 파일 수정
아래 두 줄을 실제 값으로 교체하세요const GOOGLE_CLIENT_ID = '발급받은 Client ID';const ALLOWED_DOMAIN = 'mimetics.co.kr';
Step 5. 시트 웹에 게시
Google Sheets → 파일 → 공유 및 내보내기 → 웹에 게시 → 전체 문서, CSV → 게시
✓ 설정 완료 후: Google 계정 선택 → @mimetics.co.kr 이메일 자동 검증 → 대시보드 접근
